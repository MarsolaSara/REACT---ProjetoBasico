# ⚔️ Sara Life RPG

Transforme sua rotina em uma aventura.

O **Sara Life RPG** é uma aplicação web inspirada em jogos de RPG, criada para transformar tarefas do dia a dia em missões, recompensas e conquistas. A proposta é tornar hábitos como estudar, treinar, trabalhar e passar tempo com a família mais divertidos e motivadores.

Este projeto está em desenvolvimento e será expandido para se tornar um sistema completo, com autenticação, banco de dados e back-end em Java.

---

# 🎮 Conceito

No Sara Life RPG, cada atividade realizada gera experiência (XP), fortalece atributos e aproxima o jogador de seus objetivos pessoais.

Em vez de apenas concluir tarefas, o usuário evolui seu personagem.

Exemplos:

- 🧑‍💻 Programar aumenta o intelecto.
- 🏋️ Academia aumenta a força.
- 🥊 Kung Fu desenvolve resistência.
- 🖊️ Escrever melhora a criatividade.
- 💰 Trabalhar fortalece as finanças.
- 👨‍👩‍👧 Tempo com a família melhora as relações.

# 🚀 Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/seu-usuario/sara-life-rpg.git
```

Entre na pasta:

```bash
cd sara-life-rpg
```

Instale as dependências:

```bash
npm install
```

Inicie o servidor:

```bash
npm run dev
```

O projeto será executado localmente.
